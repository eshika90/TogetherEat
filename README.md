- ☘회원가입 후 좋아하는 음식, 싫어하는 음식, 못먹는 재료 등을 선택합니다.<br>
- 🍽메뉴추천을 눌러 원하는 음식 종류의 탭을 누른 후 메뉴 추천받기 버튼 클릭!<br>

- 🍔보너스로 맛집 추천받기를 눌러 근처 음식점을 추천받을 수 있습니다!<br><br>
